## Project Setup 

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/repository-name.git
   ```

2. **Install dependencies**:
   ```bash
   cd repository-name
   npm install
   ```

3. **Run the application**:
   ```bash
   npm run dev
   ```

4. Open your browser and go to `http://localhost:5173`.
